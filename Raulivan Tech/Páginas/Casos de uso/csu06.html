Sumário: O cliente utiliza o sistema para gerenciar itens no carrinho de compras

Ator primário: Cliente

Ator secundário: Administrador

Pré-condições: O cliente deve estar logado.

Fluxo principal

  1. O cliente entra na página de carrinho;
  2. O sistema exibe uma lista com todos os produtos adicionados ao carrinho;
  3. O sistema dá a possibilidade de alterar a quantidade de cada produto e de remover um determinado produto do carrinho;
  4. O cliente aperta em comprar;
  5. O sistema inicia o processo de compra dos produtos que estão no carrinho.

Fluxo Alternativo (2): Não há produtos adicionados ao carrinho

  1. O sistema informa que não há nenhum produto para ser exibido.

Fluxo Alternativo (3): O cliente deseja alterar a quantidade de algum produto

  1. O cliente digita a nova quantidade daquele produto no carrinho;
  2. O sistema atualiza as informações do produto;
  3. O cliente aperta em comprar;
  4. O sistema inicia o processo de compra dos produtos que estão no carrinho.

Fluxo Alternativo (3): O cliente deseja excluir um produto do carrinho

  1. O cliente pressiona o botão excluir de um determinado produto;
  2. O sistema remove o item do carrinho;
  3. O sistema exibe uma lista com todos os produtos restantes adicionados ao carrinho;
  4. O sistema dá a possibilidade de alterar a quantidade de cada produto e de remover um determinado produto do carrinho
  5. O cliente aperta em comprar;
  6. O sistema inicia o processo de compra dos produtos que estão no carrinho.

Pós-condições: O sistema envia as informações do carrinho para o processo de compra que começa a ser realizado.
