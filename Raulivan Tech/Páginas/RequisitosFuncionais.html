  • Casos de uso
  • R1. O sistema deve permitir pesquisar por produtos e marcas.
  • R2. O sistema deverá permitir o gerenciamento do carrinho de compras.
  • R3. O sistema deverá ser capaz de efetuar o cadastro de clientes.
  • R4. O sistema deverá permitir consultar o historico de compras.
  • R5. O sistema deverá permitir filtrar os produtos.
  • R6. O sistema deverá permitir receber notificação.
  • R7. O sistema deverá permitir a consulta a pagina do produto.
  • R8. O sistema deverá permitir o cadastro de usuarios administrativos.
  • R9. O sistema deverá permitir o login de clientes e administradores.
  • R10. O sistema deverá permitir que administradores possam gerenciar os produtos
  • R11. O sistema deverá permitir que administradores possam alterar o status de envio de um pedido.
  • R12. O sistema deverá permitir que administradores possam realizar promoções.
