RegraNegocio00

  • Nome : Choques de horários
  • Descrição: Não será permitido que um paciente agende vacinas para o mesmo horário.

RegraNegocio01

  • Nome : Restrição de agendamento
  • Descrição: O paciente poderá agendar uma vacina somente se não houver registro desde com o processo já efetuado.

RegraNegocio02

  • Nome : Agendamento e realização de vacinação.
  • Descrição: Só será permitido agendar ou solicitar vacinação caso a vacina em questão esteja disponível no estoque.

RegraNegocio03

  • Nome : Exigir cadastro.
  • Descrição: Será permitido o agendamento de vacina somente para pacientes cadastrados.

RegraNegocio04

  • Nome : Remover do estoque.
  • Descrição: Remover todas as vacinas imediatamente do estoque após o seu vencimento.

RegraNegocio05

  • Nome : Solicitar vacinação.
  • Descrição: Será permitido ao paciente agendar previamente uma vacinação ou imediatamente em uma unidade de saúde.

RegraNegocio06

  • Nome : Comprovante de vacinação.
  • Descrição: O sistema deverá fornecer um comprovante de vacinação impresso caso o paciente solicite. Em todos os casos, logo após a confirmação de vacinação, o sistema deverá fornecer um comprovante digital que ficará disponível no sistema e será enviado para o paciente via e-mail ou mensagens (WhatsApp).

RegraNegocio07

  • Nome : Efeitos colaterais.
  • Descrição: Estará contido no sistema um registro dos sintomas mais comuns de todas as vacinas. Quando o paciente desejar relatar algum sintoma que esteja sentindo será exibido somente os efeitos mais comuns.

RegraNegocio08

  • Nome : Datas delimitadas do cadastro de campanha.
  • Descrição: Será possível realizar o cadastro de campanhas somente com 7 dias de antecedência.
